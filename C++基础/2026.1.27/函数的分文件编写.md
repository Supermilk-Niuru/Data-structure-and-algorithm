### 编写步骤：
* 创建头文件，文件后缀名必须是`.h`
* 创建源文件，文件后缀名必须是`cpp`
* 头文件中写函数的声明 **注意带上标准函数库**
* 源文件中写函数的定义 **注意带上头文件:** 目的是与头文件产生关联

---
### 怎么编译运行
* 编译
`g++ *.cpp -o 文件名`
`*`是通配符,如果觉得不麻烦的话,可以将`*.cpp`换成你需要执行的`.cpp`文件。执行完成后会产生一个二进制文件
* 运行
`./文件名`
### 使用`Makefile`简化编译运行
```
all:
    g++ *.cpp -o 文件名
    ./文件名
```
**每一个执行语句之前必须是一个`tab`**
* 如果想要清理生成的二进制文件，加上:
```
clean:
    rm -f 文件名
```
或者
```
clean:
    trash 文件名
```
`rm`不会进废纸篓,`trash`会进废纸篓,有一个缓冲过程

### 注意事项
新开的终端默认在根目录，自己定义的头文件默认在其所在文件夹有效，如果需要变化，需要执行`cd`命令语句(`change directory`)
