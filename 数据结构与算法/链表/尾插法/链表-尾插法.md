## 链表

#### 尾插法

###### 时间复杂度为$O(n^2)$
创建要插入的新节点,`next=NULL`
尾节点从头到尾遍历，直到发现`NULL`为止，此过程不断更新尾节点
找到`NULL`，更新尾节点指向新节点
* 之前的错误写法:
```C++
    for(i=0;i<n;i++)
    {
        Nodelist *newnode = new Nodelist;
        newnode->data = i*10;
        newnode->next = NULL;
        Nodelist *lastnode = head;
        if(lastnode==NULL)
        {
            lastnode = newnode;
        }
        else
        {
            while(lastnode!=NULL)
            {
                lastnode = lastnode->next;
            }
            lastnode = newnode;
        }
    }
```
1. 应该是 `lastnode->next = newnode` `lastnode->next!=NULL`
2. `Nodelist *lastnode = head;` 应该放在`else`里面 错误写法会导致每一次循环只运行`if`而不运行`else`
* 因为是**尾插法**， 所以最后结束必定是`NULL`,这也导致了插入的新节点的`next`为`NULL`
* 正确写法的核心逻辑
```C++
    for(i=0;i<10;i++)
    {
        Listnode *newnode = new Listnode;
        newnode->data = (i+1)*10;
        newnode->next = NULL;
        if(head==NULL)
        {
            head = newnode;
        }
        else
        {
            Listnode *lastnode = head;
            while(lastnode->next!=NULL)
            {
                lastnode = lastnode->next;
            }
            lastnode->next = newnode;
        }
    }
```
**注意：** 此方法构建长度为$n$的链表，`while`循环只执行$n-2$次。因为第一次循环结束后，进入`else`分支,但是不满足`while`循环的条件。循环进行到$3$次时，`while`循环开始执行
