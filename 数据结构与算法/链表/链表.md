## 链表
##### 创建链表
```C++
struct Listnode
{
    int data;
    Listnode *next;
}
```
##### 动态内存分配
```C++
Listnode *m = new Listnode;
//或者写成
Listnode *m;
m = new Listnode;
//C语言写法
m = (struct Listnode*)malloc(sizeof(struct Listnode));
```
**注意：** 必须写成`Listnode *next`，如果写成`Listnode next`会出现套娃操作，编译器无法为链表分配内存，即不断在找`Listnode`的大小。

##### 头插法
1. 新节点指向头节点
2. 头节点指向新节点(更新头)
头插法一定是**后进先出**
使用循环，不断进行内存分配，理论上长度可以不受限制。 
```C++
for(i=0;i<n;i++)
{
    Listnode *m = new Listnode;
    m->data = i * 10;
    m->next = head;
    head = m;
    //delete m;
}
```
