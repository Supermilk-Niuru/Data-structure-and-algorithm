## 链表
##### 创建链表
```C++
struct Listnode
{
    int data;
    Listnode *next;
}
```
##### 动态内存分配
```C++
Listnode *head = new Listnode;
//或者写成
Listnode *head;
head = new Listnode;
//C语言写法
head = (struct Listnode*)malloc(sizeof(struct Listnode));
```
**注意：** 必须写成`Listnode *next`，如果写成`Listnode next`会出现套娃操作，编译器无法为链表分配内存，即不断在找`Listnode`的大小。
