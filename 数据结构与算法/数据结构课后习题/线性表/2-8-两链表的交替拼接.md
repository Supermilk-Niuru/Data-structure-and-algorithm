### 题目
给两个位置长度的线性表，他们以链式的方式存储。要求：将两个表交替合并，短的表在前。
**注意**
```C++
        while(p!=nullptr)
        {
            r->next = p;
            r = p;
            r->next = q;
            r = q;
            p = p->next;
            q = q->next;
        }
```
本处`p->next`已经找不到了，因为之前`r=p;r->next=q` `p`的下一个已经指向`q`了，这就导致A链直接断掉了
```C++
        while(p!=nullptr)
        {
            Listnode *ta = p;
            Listnode *tb = q;
            r->next = ta;
            r = ta;
            r->next = tb;
            r = tb;
            p = p->next;
            q = q->next;
        }
```
本处虽然存储了，但还是会犯之前一样的错误，A链直接断掉了。
**解决方法** 存储下一个节点让链不断
