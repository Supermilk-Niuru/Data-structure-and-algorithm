### String 容器

###### string初始化
本质上就是一个赋值操作
1. C语言类型 `char[] = "   ";`
2. C++类型 `string = "    ";`
3. 利用`assign`  `str1.assign(str2);`
4. 赋相同的字符 `str1.assign(8,'a');` 表示连续8个`a`赋值
**注意：**
如果想要匹配前n个字符，那么字符串必须是C风格类型字符串。
如果是使用的是string ，则是截取赋值，从下标开始，一直到末尾。
###### string构造
**注意** 
使用字符串的时候 必须使用`const char *str = "   "`,新标准中强调不允许修改
###### string拼接
1. 使用`+=`直接添加
2. 使用`append("  ")` `append(" ",1)`将前$1$个字符添加到末尾 
`append(" ",0,2)` 从第$0$个起，添加$2$个到末尾
###### string删除插入
1.插入`insert(1," ")`从下标$1$前面开始插入
2.删除`erase(1,5)`删除下标从$1$到下标$5$的字符
###### string查找
输出值是整型，找到了输出对应值的下标，没有找到输出$-1$
###### string替换
`replace(3,4，" ")`
`replace(替换的起始位置，替换元素的个数，替换的字符串)`
###### string比较
`compare("  ")`质是逐一比较ASCII码 大 返回值 大于0 等 返回值 0 小 返回值 小于0
###### string获取字串
`substr(起始位置，截取的个数)`
###### string读和改
使用`at(下标)`
